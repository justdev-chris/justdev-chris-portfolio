## üñºÔ∏è Image 2 ASCII

**Image 2 ASCII** is a Python-based terminal tool that converts images into ASCII art.

Give it any image, and it'll turn it into a beautiful (or cursed) wall of characters ‚Äî perfect for memes, nostalgia, or command-line art shows.

### Features
- CLI-based image to ASCII converter
- Adjustable output size
- Supports most common image formats
- Outputs to console or file

### Usage
Change the input image in the python code to your image path. An output txt will come out containing the ASCII.
Optional arguments can be added for output width, contrast, and character sets depending on your version.
